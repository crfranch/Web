Summary: Latitude vs. X

For this project, I analyzed the location of various cities across the world and traced them back to their latitude. In order to accomplish this, I took the data from OpenWeatherMap, which offered an API containing the subject data.

After assembling this dataset, I was able to implement my findings into a scatter plot using matplotlib. I examined various factors of latitude's relationship on weather, including: temperature, cloudiness, wind speed, and humidity. Throughout this site, we will walk through the comparisons as we get to better know the data being examined.
